# 🌍 Meteorite Chat & Chart Bot

An interactive Flask-based web application that allows users to explore and analyze meteorite landing data using natural language queries and dynamic visualizations.

---

## 🚀 Features

- 💬 Chat-based interface to query meteorite data  
- 📊 Generate charts (bar, histogram, map, scatter) using Plotly  
- 🔍 Perform data analysis (average mass, top meteorites, counts)  
- 🌎 Location-based queries (search meteorites by region)  
- ⚡ Lightweight and fast Flask backend  

---

## 🛠️ Tech Stack

- Python (Flask, Pandas)  
- Plotly (data visualization)  
- HTML & CSS (UI design)  

---

## 📂 Dataset

- NASA Meteorite Landings Dataset  
- Includes meteorite name, mass, year, and geolocation
